# SUSMI-Hackathon-2026
SUSMI Hackathon April 2026

# Context:
Australia’s healthcare system spans public hospitals, private specialists, general practice, and allied health, often funded through Medicare, private insurance, and out-of-pocket payments. Navigating between these services can be complex, particularly in fast-growing urban centres like Sydney and in underserved regional areas.

# Your task:
How might we reduce patient drop-off at a specific point in the Australian care journey (e.g., GP → specialist, diagnosis → treatment, discharge → follow-up) by designing a solution that improves continuity, access and engagement, by addressing barriers such as cost, understanding, and system navigation?
